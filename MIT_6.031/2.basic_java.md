# 基本java
## 可变与不可变
 `String` 不可变类型，`StringBuilder` 可变的对象。  
 `final` 不可变引用，对可变值（如`StringBuilder`）进行不可变引用，其值可以更改。

## java集合
### List
```java
int count = lst.size();
lst.add(e);
if (lst,isEmpty()) ...
```
### Map
```java
map.put(key,val);
map.get(key);
map.containsKey(key);
map.remove(key);
```
### Set
```java
s1.contains(e);
s1.containsAll(s2);
s1.removeAll(s2);
```

## 快速创建数组
```java
String[] arr = {"a","b","c"};
Arrays.asList(new String[] {"a","b","c"}); //长度固定
Arrays.asList("a","b","c"); //长度固定
```

## 迭代 map
```java
for (String key : turtles.keyset()) {
    System.out.println(key + ": " + turtles.get(key));
}
```
